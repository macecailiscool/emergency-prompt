EMERGENCY PROMPT

Overview

EMERGENCY PROMPT is a Bash script that provides emergency options to Mac users in case their system becomes unresponsive or unstable. The script allows the user to choose between force restarting the Dock, Finder, or the whole system. The "Force Reboot" option includes a second warning message in red text, reminding the user that all unsaved changes will be lost.

Features

  • Three emergency options: Restart Dock, Restart Finder, or Force Reboot
  • Second warning message for Force Reboot, in red text
  • Checks whether system integrity protection is disabled
  • Logs all actions performed in a log file with a timestamp
  • Includes a warning message explaining the risks associated with using the script
  • Requires the user to confirm that they understand the risks before proceeding

Usage

  1. Download the script to your Mac
  2. Open a Terminal window
  3. Navigate to the directory where the script is located
  4. Type ./emergency-prompt.sh and press Enter
  5. Follow the on-screen instructions to choose the desired action

Note: This script is intended for use in emergency situations only, and may cause data loss or system instability. Use at your own risk.
